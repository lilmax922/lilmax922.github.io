揪遊 桌遊預約揪團網站

概念發想
本身的休閒娛樂就是與朋友去桌遊店玩桌遊，但普遍的桌遊店都沒有自己的網站能線上預約，又或是朋友都沒空但很想玩桌遊，於是將這些想法以一個桌遊網站的方式實踐。

網站介紹

揪遊是一間桌遊店，您可以在網站上查看店內擁有的桌遊及每個桌遊的詳細介紹，讓想來店內遊玩桌遊的人們可以直接線上預約，也能發起揪團或參加揪團與其他人遊玩。

  • 技術應用：Vue.js、Quasar UI、Node.js、MongoDB、RESTful API

網站&會員 功能
  • 桌遊查詢、詳細介紹
  • 註冊登入功能
  • 會員預約
  • 發起揪團、參加揪團
  • 個人資料修改
  • 取消 預約、揪團
管理者功能
  • 桌遊 增改刪查
  • 修改會員資料
  • 修改預約、揪團
